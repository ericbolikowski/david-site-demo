# WeWork Tenerife Website

A modern, responsive website for WeWork's co-working space in Tenerife, Canary Islands. This website is designed to showcase the workspace offerings and attract international digital nomads.

## Features

- **Responsive Design**: Fully responsive design that works on all devices
- **Multiple Pages**: Home, Services, Spaces, Pricing, Contact, and Order pages
- **Interactive Elements**: Modern UI with interactive components
- **PDF Contract Generator**: Live PDF generation for workspace contracts
- **Price Calculator**: Dynamic pricing based on selected plan, duration, and team size

## Technologies Used

- **Next.js**: React framework for server-rendered applications
- **TypeScript**: Type-safe JavaScript
- **TailwindCSS**: Utility-first CSS framework
- **React Datepicker**: Date selection component
- **jsPDF**: Client-side PDF generation
- **React Hot Toast**: Notification system

## Pages

- **Home**: Landing page with key features and benefits
- **Services**: Detailed information about workspace options and amenities
- **Spaces**: Gallery showcasing the different areas of the co-working space
- **Pricing**: Transparent pricing information for all membership plans
- **Contact**: Contact form and location information
- **Order**: Booking form with live contract preview and PDF generation

## Getting Started

### Prerequisites

- Node.js 18.0.0 or later
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/wework-tenerife.git
cd wework-tenerife
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Deployment

This website can be easily deployed to Vercel, Netlify, or any other hosting platform that supports Next.js applications.
